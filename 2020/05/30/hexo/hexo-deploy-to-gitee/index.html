hexo 博客部署到gitee 上

2020-05-30 Hexo 0 评论

说明

以下操作在ubuntu 系统上完成。

准备工作

  • gitee 帐号信息
  • ssh
  • git
  • hexo

配置git 的用户信息

生成ssh密钥

将生成的密钥信息配置到gitee 上

  1. 登陆码云
  2. 进入个人设置
  3. 找到安全设置–> SSH公钥
  4. 填写标题,然后将刚才 cat id_rsa.pub 这个文件里面的内容复制到 公钥输入框上
  5. 点击提交即可

测试是否链接成功

进入到hexo 文件夹上

  1. 打开_config.yml 文件

在底下找到deploy 节点 并修改你的git地址
修改时注意保证你的仓库是空的，否则hexo 上传文件时会将你原有的文件清空

  1. 安装hexo git

  2. 上传到gitee 仓库上

部署成功后进入gitee 仓库

  1. 点击服务 –> Gitee Pages
  2. 进入该页面后需要手动点击更新按钮才会部署到静态页面上
  3. 点击更新按钮后等待几分钟就可以访问了
